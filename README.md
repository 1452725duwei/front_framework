# 前端常用框架与组件库

[TOC]

## 概述

本项目整理了前端常用的布局、组件、示例页面。

要求：
1、可以涵盖大部分网站类型，包括：
* admin 后台管理
* org_portal 机构门户
* general_portal 综合门户
* news 新闻
* wiki 百科
* search 搜索引擎
* blog 个人博客
* sns 社交网络
* forum 论坛
* eshop 电子商城
* ide 在线工作台

2、可以兼容IE9以上浏览器，兼容各种分辨率的PC、平板、手机
3、代码规范遵守:  https://google.github.io/styleguide/htmlcssguide.html
